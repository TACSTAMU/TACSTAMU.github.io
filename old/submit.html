Rules

This is a great opportunity for Aggies to submit and showcase their programming skills in any coding language! The basic rules are:

  • Submissions must show some unique characteristic
  • Submissions must be a URL to a repository, preferably GitHub
  • Submissions should be independent from any school assignment

Submit

Due to our limitations of being only able to upload static content, please email us a link to your GitHub repository with your contact info and a brief description of your project that you are submitting.

Winning submissions will receieve a $25 Amazon giftcard and a spot on our main TACS webpage!
